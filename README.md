# Universal File Converter

A powerful web application to convert any file format to another – documents, images, audio, video, archives. Free, secure, and open source.

## Features

- Convert between 50+ formats
- All PDF tools (merge, split, compress, password)
- Drag & drop interface
- Real-time progress
- Automatic file cleanup

## Prerequisites

- Node.js (v16 or later)
- FFmpeg (for audio/video conversion) – [Download](https://ffmpeg.org/download.html)
